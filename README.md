# Student Course Web App

Simple Flask web application that displays students from a database and allows adding new students.

## Features
- View all registered students
- Add new students via web form
- SQLite database integration

## Tech Stack
- Python 3, Flask
- HTML, SQLite

## How to Run
1. pip install -r requirements.txt
2. python app.py
3. Open http://localhost:5000 in browser

## What I Learned
Flask routing, database connections, HTML forms, basic CRUD operations.

Live demo: Run locally with 2 commands - pip install + python app. py
